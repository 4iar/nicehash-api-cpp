5 # Nicehash API wrapper (C++)
6 
7 [![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
8 [![Build Status](https://travis-ci.org/4iar/nicehash-api-cpp.svg?branch=master)](https://travis-ci.org/4iar/nicehash-api-cpp)
9 
10 A C++ API wrapper for the [NiceHash](https://www.nicehash.com/) cloud cryptomining service.
11 
12 ## Requirements
13 
14 - cURL libs (e.g. Fedora: libcurl-devel, Ubuntu/Debian: libcurl-dev)
15 
16 - CMake >= 3.1
17 
18 ## Setup
19 
20 See example CMakeLists.txt in the [examples/](https://github.com/4iar/nicehash-api-cpp/tree/master/examples) folder that uses
21 [ExternalProject](https://cmake.org/cmake/help/v3.2/module/ExternalProject.html) to integrate nicehash-api-cpp with an existing CMake project.
22 
23 Alternatively, manually build the library as below, then link libnicehash-api-cpp.so.
24 
25 ```bash
26 git clone https://github.com/4iar/nicehash-api-cpp.git && cd nicehash-api-cpp
27 mkdir build && cd build
28 cmake ..
29 make
30 ```

59 #### Global
60 
61 ##### stats.global.current
62 
63 Get current profitability (price) and hashing speed for all algorithms. Refreshed every 30 seconds.
64 
65 `niceHashApi.getAverageGlobalStats()`
66 
67 ##### stats.global.24h
68 
69 Get average profitability (price) and hashing speed for all algorithms in past 24 hours.
70 
71 `niceHashApi.getCurrentGlobalStats()`
72 
73 #### Provider
74 
75 ##### stats.provider
76 
77 Get current stats for provider for all algorithms. Refreshed every 30 seconds. It also returns past 56 payments.
78 
79 `niceHashApi.getProviderStats (std::string address)`
80 
81 ##### stats.provider.ex
82 
83 Get detailed stats for provider for all algorithms including history data and past 56 payments.
84 
85 `niceHashApi.getDetailedProviderStats (std::string address)`
86 
87 ##### stats.provider.workers
88 
89 Get detailed stats for provider's workers (rigs).

91 `niceHashApi.getProviderWorkerStats (std::string address, int algorithm_id)`
92
